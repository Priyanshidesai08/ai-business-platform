# Demo Script

## 1. Start the Stack
```bash
docker compose up --build
```

## 2. Register a User
- Open `http://localhost:5174/register`
- Fill the form and submit

## 3. Log In
- Open `http://localhost:5174/login`
- Log in with the new account

## 4. Show the Dashboard
- Confirm the dashboard loads
- Point out agent cards and protected navigation

## 5. Test a Business Module
- Open Sales, Marketing, Support, or Analytics
- Show live data and workspace actions

## 6. Show Workflow Builder
- Open Workflow Builder
- Add or select nodes
- Save, restore, run, pause, resume, and stop

## 7. Show Monitoring
- Open Monitoring
- Review KPI cards, ROI panel, and feedback capture

## 8. Show Swagger
- Open `http://localhost:5001/api-docs`
- Run a documented endpoint

## 9. Closing Summary
- Mention Docker, PostgreSQL, JWT auth, Swagger, AI orchestration, workflows, and monitoring
